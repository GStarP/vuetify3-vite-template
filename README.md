# vuetify3-vite-template

a simple but complete template for Vuetify app development.

## Features

- TypeScript
- Vite
- Vue 3
- Vuetify
- Vue-Router
- Pinia
- Prettier
- MockJS
- Electron (optional)

## Quick Start

serve web app:

```bash
yarn install
yarn dev
```

serve electron and load web app:

```bash
yarn e  # must be run after `yarn dev`
```
